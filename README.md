# China Insider

China Insider is an English-first web app for matching foreign travelers with vetted local experts in China for deep-interest cultural experiences.

## V1 direction
- Founder-operated marketplace MVP
- Text-first concierge
- Strong extensible foundation for V2
- Realtime voice, embeddings, expert self-serve, and payout automation are intentionally deferred

## Stack
- Next.js 14 App Router
- React 18
- TypeScript strict mode
- Tailwind CSS + shadcn/ui
- Supabase
- Stripe
- Vitest
- Biome

## Available scripts
- `pnpm dev`
- `pnpm test`
- `pnpm typecheck`
- `pnpm biome:check`
- `pnpm build`
